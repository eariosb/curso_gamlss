## 2. GAMLSS bayesiano

El enfoque frecuentista de `gamlss()` estima parámetros por máxima
verosimilitud penalizada y usa aproximaciones asintóticas para los errores
estándar. El enfoque **bayesiano** coloca priors sobre los parámetros y
obtiene la distribución posterior completa, con intervalos creíbles que no
dependen de grandes muestras.

### Opción 1: brms (Stan)

El paquete `brms` (Bürkner, 2017) permite ajustar modelos distribucionales
usando Stan (Hamiltonian Monte Carlo):

<CodeBlock
  id="20/brms-distribucional"
  title="Modelo distribucional bayesiano con brms"
  code={`library(brms)

severidad_acc <- read.csv("data/severidad_acc.csv")

# Modelo BCT bayesiano: mu y sigma con covariables
# brms soporta families exponenciales y algunas personalizadas
m_bayes <- brm(
  bf(severidad ~ log(monto) + sector + antiguedad,
     sigma ~ sector),
  family = Gamma(link = "log"),
  data = severidad_acc,
  chains = 4, cores = 4, iter = 2000,
  prior = c(
    prior(normal(0, 5), class = "b"),
    prior(normal(0, 2), class = "b", dpar = "sigma")
  )
)

summary(m_bayes)
plot(m_bayes)  # traceplots y posteriores

# Intervalos creibles del 95%
posterior_summary(m_bayes, probs = c(0.025, 0.975))`}

<Callout type="tip" title="¿Cuándo usar bayesiano?">

- **Incertidumbre en cuantiles**: el VaR 99.5% tiene un intervalo creíble
  natural, no una aproximación delta-method.
- **Priors informativos**: cuando hay conocimiento experto (ej. "la PD no
  puede exceder 50%"), los priors lo incorporan formalmente.
- **Modelos jerárquicos complejos**: cuando hay muchos niveles de efectos
  aleatorios, HMC converge mejor que RS.
- **Costo**: más lento (minutos vs. segundos) y requiere Stan.

</Callout>

### Opción 2: gamlss.bian

El paquete `gamlss.bian` extiende `gamlss()` con inferencia bayesiana usando
INLA (Integrated Nested Laplace Approximation), más rápida que MCMC para
modelos con suavizados:

```r
library(gamlss.bian)
m <- gamlss(severidad ~ pb(log(monto)) + sector,
            family = BCT, data = df, method = "INLA")
```

## 5. El viaje recorrido

En 20 módulos hemos pasado de:

- **"La media no basta"** (Módulo 1) a un **sistema completo de provisiones
  IFRS 9** (Módulo 19).
- **Una sola distribución** (Normal) a un **catálogo de 100+ familias**.
- **Modelar solo $\mu$** a **modelar $\mu$, $\sigma$, $\nu$ y $\tau$**
  simultáneamente.
- **GLM/GLMM** a **GAMLSS con efectos aleatorios, suavizados y censura**.
- **Seguros** a **inventarios, calidad, materiales, biología y finanzas**.

El hilo conductor: **la distribución completa importa**. No solo la media,
no solo la varianza, sino la forma entera de la distribución condicional de
$Y$ dado $\mathbf{x}$. Esa es la esencia de GAMLSS y el lema de este curso:
*rage against the mean*.
